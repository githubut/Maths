Method 03
Near 100 Base Method

"For numbers near 100, use this pattern: (N + 2d) | 3d² | d³. It uses the deviation (d) from 100 to build the cube in three clean chunks."

Problem: 102³
1. N + 2d
102 + 4 = 106
Deviation (d) is 2. So 102 + (2x2) = 106.
2. 3 × d²
3 × 4 = 12
Triple the square of the deviation. 3 x 4 = 12.
3. d³
2³ = 08
Cube the deviation. (Must be 2 digits, so 08).

Method 04
Near 50 Base Method

"When near 50, we use 125,000 as our base (since 50³ = 125k). Add 7,500 for every unit above 50, then 150 for every unit squared. It’s a bit of addition but far faster than long multiplication!"

Problem: 52³
1. Base Offset
125,000 + (2 × 7,500)
140,000
50³ is 125k. Each step from 50 adds roughly 7,500 to the volume.
2. Second Correction
150 × 2² = 600
Triple the base (50) times the deviation squared. 150 × 4.
3. Final Cube
2³ = 8
Simply add the cube of the deviation at the end. (140,000 + 600 + 8)
